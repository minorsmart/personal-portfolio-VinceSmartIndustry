Smart Technology

Tijdens Smart Technology hebben we een applicatie ontwikkeld m.b.v. een IOT Device (ESP32) Van het IOT device is een PCB (printed circuit board) ontworpen en gefabriceerd. De opstelling is gerealiseerd m.b.v. 3d printing weken is een workshop gevolgd over 3D printen

  • Aansluiten hardware
  • Ontwerpen PCB
  • 3D printen

Zelf ontwikkelend project

Ik merkte al vrij snel tijdens de workshops Smart Technology dat ik geinteresseerd was om mijn eigen project te maken. Dit had ermee te maken dat ik web development steeds leuker begon te vinden en erachter kwam dat de ESP32 een eigen interne webserver. Via het project heb ik al verschillende communicatie technologieën onderzocht en dit wou ik ook graag in de praktijk gebruiken.

Ik stelde de volgende eisen aan het project:


  • Met Wi-Fi een webpage via de ESP32 Hosten

    • Met behulp van de WIFI.h library is dit gelukt. Een extern .h c+ bestand is gekoppeld waarin met behulp van CSS en HTML vorm gegeven is aan de webpage

  • Analoge waardes (temperatuur,vochtigheid,droogte,licht intensiteit) uitlezen en automatische waardes op te halen naar de webpage zonder te refreshen

    • Dit was erg uitdagend, bij het begin is geprobeerd om 1 analoog signaal te versturen. Het automatische ophalen(zonder F5 te gebruiken) bleek erg lastig te zijn en na wat onderzoek is het AJAX protocol gebruikt. Dat is een object die gebruik maakt van xml en Http en ervoor zorgt dat bepaalde gedeeltes van de site automatische gerefresht worden.

    Nadat 1 analoog signaal gelukt was moesten er nog 4 toegevoegd worden. Omdat er maar 1 kanaal/ object was en ik niet wist hoe ik meerdere http kanalen kon openen heb ik 1 grote JSON formaat string verstuurd en die weer opgesplitst in de data die ik nodig had.

  • Digitale signalen versturen om het klikaanklik uit systeem te bedienen.

    • Hiervoor is het SwitchKaku library gebruikt en met het bedienen van de buttons wordt een HTTP request gedaan die dit mogelijk maakt.

  • Simpele visuals / afbeelding op de webserver krijgen

    • Er is een afbeelding gehost die een schematische weergave van de smart plant laat zien.

  • Bereikbaar via verschillende webclients (meerdere gebruikers)

    • Ik wou via m'n laptop en mobiel de webpage tegelijkertijd kunnen benaderen

Competenties

  • Past nieuwe technologieën op het gebied van van IoT, Robotica en 3D-printen toe op een praktijkcasus (ST21),plaatst de casus in een business context (ST22)

    • Er is een 3D tekening gemaakt in Sketch-up van een bakje voor de behuizing van het project. Ook heeft het project veel IOT geralteerde technieken in zich zoals webhosting en Wireless communicatie

  • Vertaalt een praktijkcasus naar een onderzoeksplan, voert deze uit en evalueert de kwaliteit van uitvoering (ST61)

    • In het project is er in een praktijk omgeving gewerkt met de nieuwste technologieën. Denk hierbij aan het communicatie technologie OPC UA. Er is ook veelvuldig contat geweest met verschillende partijen om een grondig onderzoek te krijgen

  • Communiceert op effectieve wijze zijn bevindingen naar verschillende doelgroepen (ST71)

    • Het internet is vaak geraadpleegd om informatie te krijgen die nodig was om de webserver te hosten vanuit de ESP32. Ook heb ik als student met een electrotechniek achtergrond regelmatig mensen geholpen die daar behoefte aan hadden.
