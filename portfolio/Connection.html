Smart Connection

In de Smart Connection lessen hebben we les gehad in de technieken van web development en relevante technieken. Deze site is tevens gemaakt met de technieken die tijdens de les aan bod zijn gekomen

Activiteiten

  • HTML

    • HTML staat voor Hyper text markup Language en is de standaard opmaak taal voor webpagina's. De inhoud van deze site is m.b.v. HTML gemaakt. Bij het begin van de minor was ik nog niet erg bekend met HTML.

  • CSS

    • CSS staat voor Cascading sheet styles en is een taal om een webpagina mee vorm te geven. De vormgeving van deze site (kleuren, positionering en structuur is gedefineerd in het css bestand wat gekoppeld is aan deze html pagina.

  • JavaScript

    • Javascript is een scripttaal en wordt veelvuldig gebruikt om interactieve elementen toe te voegen aan een webpagina. Denk hierbij aan het connecten met een database en het bewerken van inhoud op de site. Er is voor het project van Technology verschillende scripting gebruikt. Dit om de analoge waardes van de sensoren naar de webpage te sturen.

  • Database

    • Wanneer het gewenst is om data een langdurige tijd op te slaan, denk hierbij aan klantengegevens in een klantregister. Is een Database cruciaal om met een database te communiceren wordt er een API of library gebruikt zoals ODBC en JDBC

    Zie het overdrachts document paragraaf 2 voor meer informatie over databases.

    • Overdrachts document

  • REST API

    • Rest(Representational state transfer) is een software architectuur voor het wereldwijde web en wordt vaak gebruik in combinatie met HTTP zie Uitleg communicatie protocollen. Tijdens het project was het belangrijk om een goed beeld te krijgen van API aangezien we daar vermoedelijk mee moesten gaan communiceren

  • Big Data and machine learning

    • In een workshop is machine learning behandelt. Dit aan de hand van een college met een tool die aan de hand van een foto de inhoud ervan kon benoemen.

Competenties

Tijdens het vak Smart Connection heb ik gewerkt aan de verschillende competenties

  • Past nieuwe technologieën op het gebied van web, data, cloud en AI toe op een praktijkcasus (SC21),plaatst de casus in een business context (ST22)

    • Tijdens de lessen van Smart Communication zijn er verschillende nieuwe technologieen aan bod gekomen. De oefeningen en workshops die gevolgt zijn hebben er toe geleidt dat er een beeld is van de nieuwe technologieën en ook toegepast kunnen worden.

  • Vertaalt een praktijkcasus naar een onderzoeksplan, voert deze uit en evalueert de kwaliteit van uitvoering (SC61)

    • Voordat ik aan de minor begon deed ik eigenlijk alles via word/tekst bestanden inleveren. Maar nu heb ik de vaardigheden geleerd om web-based te werk te gaan. Dit geeft een hoop voordelen aangezien het makkelijk gedeeld kan worden met de gehele wereld. Veel werk wat ingeleverd is heb ik ook laten nakijken door mede studenten dit om de kwaliteit te waarborgen

  • Communiceert op effectieve wijze zijn bevindingen naar verschillende doelgroepen (SC71)

    • Er zijn animatie filmpjes gemaakt,infographics,websites, en vlogs. Dit zijn allemaal vormen van visueel informatie overbrengen die steeds vaker en slimmer ingezet worden.
